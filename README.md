# Piedmont-IT Simple Website

## Azure Deployment
1. Create a GitHub repository.
2. Upload all files.
3. In Azure create a Static Web App.
4. Connect GitHub repo and branch main.
5. App location: /
6. API location: leave blank.
7. Output location: leave blank.

## Custom Domain
Add custom domain piedmont-it.com and www.piedmont-it.com in Static Web App > Custom domains.
Create required DNS records in your DNS provider. Configure both apex domain and www host. Azure will issue certificates automatically after validation.
